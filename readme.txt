Description :
Script "information.sh" is a menu driven program to do the following : 1. Change Password 2. See the disk space 3. Login to other box using ssh 4. All Service running 5. Show open ports 6. Show all java apps running 7. Facility to kill a app 8. Exit

FILE Description :
1.information.sh This is the script to execute above tasks.


How to run : copy the script to any directory and run it as ./information.sh or with full system path.You should be super user to run the above script. Many systems commands need root access
